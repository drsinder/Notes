PrintDlg Class

Namespace: Notes.Client.Dialogs
Assembly: Notes.Client (in Notes.Client.dll)
This takes the provided PrintStuff string and puts it in a Syncfusion editor component for the user to see. It then allows the user to press a button to print. TO do the printing the built in print function of the Sysncfusion editor is used so printing does not have to be separately implemented.
