Remarks
 
This partial class is designed to be used as a Blazor component for exporting notes, handling export formatting, file saving, and email sending. It integrates with JavaScript interop for file operations and leverages Blazored Modal for user interaction. The class manages export state and lifecycle events, and interacts with various models and services to perform export tasks. Thread safety is not guaranteed; instances are intended for use within the Blazor component lifecycle.
